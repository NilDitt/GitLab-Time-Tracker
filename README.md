# GitLab Time Tracking Dashboard

A comprehensive Next.js dashboard for visualizing GitLab project time tracking, commit activity, and team collaboration.

- **Dynamic Time Tracking**: Real-time data from GitLab timelogs and issues
- **Epic-based Analytics**: Weekly breakdown by epic with interactive charts

- **Team Collaboration**: Automatic contributor discovery from GitLab
- **Interactive Charts**: SVG-based charts with hover tooltips and filtering

## ğŸš€ Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/NilDitt/GitLab-Time-Tracker.git
   cd Random
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Configure GitLab access**

   - Copy the example environment file:
     ```bash
     cp .env.example .env.local
     ```
   - Open `.env.local` and add your GitLab Personal Access Token:

     ```env
     NEXT_PUBLIC_GITLAB_TOKEN=your-gitlab-token-here
     ```

     > **Note:** The token requires `read_api` scope.

   - (Optional) Edit `app/config/tracker-config.ts` to update the `PROJECT_PATH` if you are tracking a different project.

4. **Start development server**

   ```bash
   npm run dev
   ```

5. **Open in browser**
   - Navigate to [http://localhost:3000](http://localhost:3000)
